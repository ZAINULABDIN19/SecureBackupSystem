# Secure Backup System

A Python-based **Information Security project** with GUI for secure file backups.

## Features

- User registration and authentication with role-based access (admin/user)
- AES encryption for secure backups
- RSA encryption for key management
- List, create, and restore backups securely
- Tkinter GUI for easy interaction

## Technologies Used

- Python 3
- Tkinter (for GUI)
- Cryptography library (AES & RSA encryption)
- JSON (for user data storage)

## Usage

1. Clone the repository:

```bash
git clone https://github.com/ZAINULABDIN19/SecureBackupSystem.git

2. Install dependencies:
pip install cryptography

3. Run the GUI version:
python gui_app.py

4. Or run the CLI version in Jupyter Notebook
jupyter notebook main.ipynb
